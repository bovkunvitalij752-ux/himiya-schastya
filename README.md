# Химия счастья

Современная AI-платформа для анализа взаимоотношений, диагностики отношений и поддержки крепких семей.

## Технологии

- Next.js
- TypeScript
- Tailwind CSS
- App Router
- Supabase
- Vercel

## Запуск

```bash
pnpm install
pnpm dev
```

## Проверки

```bash
pnpm typecheck
pnpm lint
pnpm build
```
